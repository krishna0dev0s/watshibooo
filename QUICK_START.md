## üéØ Resume Sections Explained

### **Contact Information** (Required)
```
Name
Email
Phone
Location
LinkedIn (optional)
```

### **Professional Summary** (Recommended)
```
2-4 sentences highlighting:
- Your expertise
- Key achievements
- Unique value
```

### **Skills** (Important for ATS)
```
Organize by category:
- Technical Skills
- Leadership Skills
- Tools & Platforms
- Languages
```

### **Experience** (Essential)
```
For each position:
Job Title
Company | Duration
- Bullet point with achievement
- Include metrics (numbers, %)
- Use action verbs
```

### **Education** (Recommended)
```
Degree
School/University
Graduation Date
GPA (if 3.5+)
```

**Q: Can I change templates later?**
A: Yes! Click "Change Template" anytime to switch.

A: No! All templates are 100% ATS-optimized.

**Q: Which template should I use?**
A: Choose based on your industry (see table above).

**Q: Can I get multiple resume versions from AI?**
A: You now get 1 best version (no multiple options).
